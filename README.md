# learn-vue
some examples to practice Vue

## Install (MacOS)

* Install NodeJs: go to [Node Js Website](https://nodejs.org/en/download/) and download appropriate version
* Install Vue: open terminal and run `sudo npm i -g vue-cli eslint`
